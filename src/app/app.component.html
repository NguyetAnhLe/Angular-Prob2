Ex 3.

Components vs Directive

Component is a basically a directive with an associated view, mean while directive doesn't have view. All components are directives, but not all directives are components. Directive is usually to add behavior to a DOM element. Many directive can be used per DOM element, while only one component can be present per DOM element.

OnInit vs Constructor

The Constructor is a default method of the class that is executed when the class is instantiated and ensures proper initialisation of fields in the class and its subclasses.
Called first time before the ngOnInit()

ngOnInit is a life cycle hook called by Angular to indicate that Angular is done creating the component.
Called after the constructor and called after the first ngOnChanges().
